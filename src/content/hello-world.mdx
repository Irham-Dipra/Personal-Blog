Welcome to my new personal blog! This site is designed with a strict black and white minimalist aesthetic, focusing entirely on typography and content.

## Why this stack?

I chose **Next.js** for its incredible performance and developer experience. By using `next-mdx-remote`, I can write all my posts in standard Markdown (MDX) and have them converted into fast, static pages.

Here are a few reasons this setup is great:

1. **Blazing Fast**: Since pages are statically generated, they load instantly.
2. **Custom Components**: MDX allows me to embed React components right inside markdown!
3. **Sleek Styling**: Using Tailwind CSS, I configured a custom theme that respects both light and dark mode preferences while staying strictly monochrome.
